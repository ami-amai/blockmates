# Miness

## Описание

* Настольная ванильная Minecraft игра про поочередную установку блоков. Побеждает игрок с наибольшим количеством очков после последнего поставленного блока

## Необходимое

* [**Блоки**](./BLOCKS.md) (2 набора)
* Шерсть (2 цвета)
* Поле
* Таблички с именем и счетом
* Рамки

## Игровой процесс

### Подготовка к игре

Перед началом партии каждый игрок получает **идентичный набор блоков** согласно выбранному варианту правил. Игроки выбирают цвет шерсти — это будет маркер их блоков на поле.

### Ход игрока

Каждый ход состоит из следующих этапов:

#### 1. Размещение маркера
Игрок выбирает **свободную клетку** на игровом поле и устанавливает туда:
- Шерсть своего цвета (в качестве маркера)
- Рамку поверх шерсти

#### 2. Установка блока
Игрок выбирает **один блок** из своего набора и помещает его в установленную рамку. После использования блок убирается из набора игрока.

#### 3. Подсчёт результатов хода
После установки блока необходимо:
- Проверить, какие блоки **разрушаются**
- Подсчитать **очки**, полученные или потерянные за этот ход
- Записать **обновлённый счёт** на табличку

> **Важно:** Блоки взаимодействуют согласно [правилам](./BLOCKS.md#взаимодействие).

### Чередование ходов

Игроки ходят **по очереди**, каждый раз устанавливая по одному блоку из своего набора. Пропускать ход нельзя.

### Завершение игры

Игра **заканчивается**, когда устанавливается **последний блок**.

### Определение победителя

**Побеждает игрок**, набравший **наибольшее количество очков** к моменту окончания игры.
